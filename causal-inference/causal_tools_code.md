不定期更新相关内容，用于学习交流 :) 1.Dowhy： 微软出品。基于因果推理统一语言，结合了因果图模型以及潜变量结果框架，支持因果推断的结构建模和应用。 官网：GitHub - microsoft/dowhy 2.CausalDiscoveryToolbox(CDT): 简介：基于神经网络 CGNN 的图形化（networkx）因果推断。基于NumPy，sklearn，PyTorch，bnlearn, 和 pcalg。 官网：GitHub - GoudetOlivier/CausalDiscoveryToolbox 3.causal-learn 简介：CMU张坤老师出品，容易上手实践的一款工具。 官网：GitHub - cmu-phil/causal-learn 4.珠算 简介：清华大学朱军老师出品。支持贝叶斯深度学习。 官网：GitHub - thu-ml/zhusuan 5\. causalml 简介：uber出品。基于机器学习的因果推断的综合包。主要功能包括：基于实验数据，计算CATE和ITE；基于决策树的算法；Meta-learner algorithms；工具变量算法；基于神经网络的算法。 官网：GitHub - uber/causalml: Uplift modeling and causal inference with machine learning algorithms 6\. EconML 微软出品。 简介：机器学习因果推断综合包，对标 causalml，安装相对容易一些。主要功能：主要用于计算HTE异质性处理效应；提供模型的解释，以及系数的CI置信区间；基于观测数据，进行因果推断。 官网：GitHub - microsoft/EconML
